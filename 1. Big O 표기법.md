* 빠른것?

#### Example

* for문 5n

  ```javascript
  function addUpToFirst(n) {
      var total = 0;
      for (var i = 0; i <= n; i++){
          total += i;
      }
      return total;
  }
  ```

* 곱셈 3

  ```javascript
  function addUpToSecond(n) {
  	return n * (n+1) / 2;
  }
  ```

위의 예를 보면 for문을 사용하게 되면 숫자가 증가하게 되면 그에 따라 연산 횟수도 많아지지만 곱셉을 이용한 방식은 아무리 숫자가 커져도 연산 횟수는 3으로 고정 되어있다.
